nanover.ase.imd_calculator moduleÔÉÅ

Provides an implementation of IMD force field in ASE.

class nanover.ase.imd_calculator.ImdCalculator(imd_state: ImdStateWrapper, imd_force_manager: ImdForceManager | None = None, calculator: Calculator | None = None, atoms: Atoms | None = None, dynamics: MolecularDynamics | None = None, reset_scale=0.5, **kwargs)ÔÉÅ

Bases: Calculator

Implementation of IMD as an ASE calculator.

Given another ASE calculator to act as the internal calculator, will compute the external energy and forces via the IMD service, and add them to the internal force calculations.

Parameters:

  • imd_state ‚Äì A wrapper that provides access to the active interactive forces.

  • calculator ‚Äì An existing ASE calculator to perform internal energy calculation.

  • atoms ‚Äì An ASE atoms object to use.

  • dynamics ‚Äì An ASE dynamics object from which to draw the equilibrium temperature for resetting velocities

  • reset_scale ‚Äì A scale factor to apply to velocities when reset.

  • kwargs ‚Äì Key word args passed to the base calculator.

See also

The ImdServer class makes use of this class, and makes running an interactive molecular dynamics simulation in ASE straightforward.

Calculates the given properties of the ASE atoms. The internal molecular calculator is called first, and then any interactive forces currently being applied to the system are added.

Results are stored in the results dictionary, as normal.

Parameters:

  • atoms ‚Äì Optional Atoms object to perform the calculation on. If no atoms is passed, the atoms object passed at initialisation are used.

  • properties ‚Äì The properties to calculate. The ImdCalculator support ‚Äòenergy‚Äô and ‚Äòforces‚Äô, but will pass any other requested properties to the internal atomic calculator. See calculate() for details.

  • system_changes ‚Äì List of what has changed since last calculation. See calculate() for details.

Raises:

ValueError ‚Äì If no ASE atoms are supplied to the calculation, and no ASE atoms were supplied during initialisation.
