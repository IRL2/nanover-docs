run(steps: int | None = None, block: bool | None = None, reset_energy: float | None = None)ÔÉÅ

Runs the molecular dynamics.

Parameters:

  • steps ‚Äì If passed, will run the given number of steps, otherwise will run forever on a background thread and immediately return.

  • block ‚Äì If False run in a separate thread. By default, ‚Äúblock‚Äù is None, which means it is automatically set to True if a number of steps is provided and to False otherwise.

  • reset_energy ‚Äì Threshold of total energy in kJ/mol above which the simulation is reset to its initial conditions. If a value is provided, the simulation is reset if the total energy is greater than this value, or if the total energy is nan or infinite. If None is provided instead, then the simulation will not be automatically reset.
