nanover.mdanalysis.recordings module

exception nanover.mdanalysis.recordings.InvalidMagicNumber

Bases: Exception

The magic number read from a file is not the expected one.

The file may not be in the correct format, or it may be corrupted.

class nanover.mdanalysis.recordings.Unpacker(data: bytes)

Bases: object

Unpack data representations from a buffer of bytes.

The unpacking methods return the requested value from the next bytes of the buffer and move the cursor forward. They raise an IndexError if the buffer is too short to fullfil the request.

unpack_bytes(n_bytes: int) → bytes

Get the next n_bytes bytes from the buffer.

The method raises a ValueError if the requested number of bytes is negative.

unpack_u128() → int

Get an unsigned 128 bits integer from the next bytes of the buffer.

unpack_u64() → int

Get an unsigned 64 bits integer from the next bytes of the buffer.
