Tutorials

This section provides tutorials on getting started with NanoVer. We assume familiarity with setting up simulations in whichever framework you’re running with NanoVer.

The tutorials use Jupyter notebooks, NGLView for visualising trajectories, and while not strictly necessary, assumes you have the NanoVer IMD application installed (i.e. the VR client). These can all be installed with conda:

# Assuming NanoVer is installed in a conda environment named "nanover",
# change the name otherwise.
conda activate nanover
conda install jupyter
conda install nglview
# On Windows only:
conda install -c irl nanover-imd

If you wish to access the Jupyter notebooks via JupyterLab, installing JupyterLab before NGLView should install the correct dependencies for NGLView automatically (i.e. replacing conda install jupyter with conda install jupyterlab in the installation instructions above). If JupyterLab is installed outside of your conda environment, you will need to install ipywidgets in your NanoVer conda environment following these instructions.
